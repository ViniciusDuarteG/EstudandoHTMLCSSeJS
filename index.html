Como surgiu a internet

(Voltar)

A internet começa com a computação, e a computação teve seus primeiros passos com Alan Turing, na Segunda Guerra Mundial, onde a computação foi necessária para teder os ataques dos nazistas, após isso Tim Berners-Lee iniciou os primeiros passos do que podemos chamar de internet. Como funciona o HTML

(Voltar)

O HTML é um marcador de texto para navegadores da internet (Browsers), sua funcionalidade é estruturar o site, ou seja, é como se fosse as vigas de uma construção, oque da a base e a estrutura para que tudo seja incluido no site. Seja imagens, titulos, textos, videos ou até mesmo links externos, o HTML é responsável por ordenar e estruturar tudo de maneira hierarquica.

O que são Tags

(Voltar)

Tags são os marcadores que ajudam a estruturar e criar um hierarquia dentro do HTML, como por exemplo a TAG <h1≷ que serve para identificar qual é o titulo principal, ou a tag <p≷ para paragrafos

Além dos marcadores de texto também á marcadores de mídia, como audios, imagens e videos. Para uma imagem você pode usar <img≷, para audio <audio≷, e para video <video≷

Como o HTML é um marcador de texto, não podemos deixar os caracteres especiasi de fora, segue uma pequena lista dos caracteres especiais
